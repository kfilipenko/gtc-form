# RJAKA media upload path

Папка для медиафайлов новых страниц:
- Локально на сервере: `/var/www/gtc-form/assets/rjaka/media/`
- Публичный базовый URL: `https://app.gtstor.com/assets/rjaka/media/`

## Как публиковать

1. Загружайте файлы в эту папку (jpg/png/webp/mp4).
2. Используйте прямые ссылки на страницах:
   - `https://app.gtstor.com/assets/rjaka/media/<имя_файла>`
3. После переноса домена заменится только корень:
   - `https://rjaka.pro/assets/rjaka/media/<имя_файла>`

## Имена файлов (рекомендуется)

- Только латиница, цифры и дефис.
- Примеры:
  - `hero-cover.webp`
  - `rules-step-1.jpg`
  - `teaser-01.mp4`
