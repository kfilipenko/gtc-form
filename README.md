# gtc-form
Форма для регистрации на ProdAn
Создана для регистрации пользователей с проверкой повторения @адресов. 
Данные формы передаются во внешнее хранилище GT
Новая запись передается в WF на n8n

## Google onboarding (chat application)

В рамках нового чата реализуется отдельная вкладка «Google & Enterprise», которая берет на себя весь OAuth‑флоу:

- Рендер кнопки Google через GIS внутри самого `chat/index.html`, без перехода на устаревшую страницу `/auth/google/`.
- Получение `credential` происходит в попапе; результат обрабатывается на той же странице и отправляется на `/auth/google` через `fetch`.
- После успешного ответа сразу обновляется локальное состояние (`state.user`, `localStorage`, гидратация профиля, проверка биллинга) и открывается чат.
- Для браузеров с блокировкой попапов остаётся fallback‑кнопка, но она ведёт на лёгкий промежуточный холст, который возвращает токен обратно в чат через `postMessage`.
- На вкладке отображаются вспомогательные сервисы: статус верификации, привязка тарифа, диагностика доступа.

После подтверждения дизайна вкладка и сервисы внедряются непосредственно в `chat/index.html`, чтобы пользователь больше не покидал чат во время регистрации через Google.
