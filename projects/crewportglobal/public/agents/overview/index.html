Participant section

Agents

This section is for agents and agent companies that work for represented seafarers or shipowners under verified authority and controlled platform assignments.

Role in the system The agent is a representative and operational facilitator. The agent may prepare work for both sides, but platform management requires authority, agreement, one active manager and participant governance visibility.

Why this page exists

Use this page to understand the agent role before opening the operational agent portal. The platform separates explanation, authority and live scoped work.

Basic order of actions

  1. Register as a physical person and request or confirm agent capacity.
  2. Connect an agent organization and provide authority evidence.
  3. Receive or create represented-object work only inside verified platform scope.
  4. Accept shipowner or seafarer representation through platform agreements, with commercial terms agreed separately where needed.

Important boundary

An agent may assist both sides in finding, preparing and coordinating work, but cannot silently replace the personal governance rights of the represented participant.
