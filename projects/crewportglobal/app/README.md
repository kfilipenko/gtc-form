# CrewPortGlobal Application Shell Skeleton

## Purpose

This directory is a planning-only skeleton for CPG-I1-001, Website application shell planning.

It exists only to record the approved top-level application shell split for the CrewPortGlobal Increment 1 seafarer-only prototype.

## Approved Baseline

1. CrewPortGlobal website application runtime: GTC1
2. CrewPortGlobal SQL database locality: GTC1
3. OpenClaw runtime / agent platform: GTC-AGENT
4. OpenClaw usage: assisted operator support only, through controlled procedures

## Approved Directory Scope

This skeleton is limited to:

1. frontend/
2. backend/
3. shared/

## Planning Boundary

This directory does not authorize or include:

1. application code
2. package.json
3. vite.config or next.config
4. tsconfig
5. src, routes or api implementation
6. server runtime files
7. .env files
8. Docker or compose files
9. nginx or systemd configuration
10. SQL migrations
11. OpenClaw configuration
12. deployment artifacts

## Control Statement

Application shell skeleton is prepared for project-owner review only. Implementation execution remains not approved.
