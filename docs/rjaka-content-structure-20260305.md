# RJAKA — Структурированная контент-карта (2026-03-05)

Источник данных:
- Внешние страницы: `/buy`, `/rules`, `/rgame`, `/meaning`, `/game-card` на `rjaka.pro`.
- Контент-файл: `RJAKA_CORE.md`.

## 1) Купить игру (`/buy`)

Цель страницы:
- Конверсия в покупку + быстрый переход в чат-помощник.

Смысловые блоки:
- Что такое игра (кратко, 1–2 абзаца).
- Что входит в коробку.
- Для кого подходит.
- CTA на покупку/чат.

Контент из `RJAKA_CORE.md`, который использовать:
- «Что в коробке?»
- «Где купить игру?»
- Короткое описание формата игры.

## 2) Правила (`/rules`)

Цель страницы:
- Быстрый вход в игру без перегруза.

Смысловые блоки:
- Как это работает: «Роль → Действие → Сценка → Улыбки».
- Главный закон: без критики, только поддержка.
- Как проходит ход.
- Когда заканчивается игра (3 варианта).
- Режимы: «Быстрый старт», «Роли в руках».

Контент из `RJAKA_CORE.md`, который использовать:
- Разделы правил и инфографические схемы.

## 3) Об игре (`/rgame`)

Цель страницы:
- Объяснить ценность и формат бренда/игры.

Смысловые блоки:
- Что это за игра и чем отличается.
- Где используется (дом, друзья, семьи, команды).
- Герои/роли как ядро механики.
- Быстрый переход к картам ролей и чату.

Контент из источников:
- Заголовок «Герои Игры Ржака».
- Описание бренда и применения (из `meaning` + core).

## 4) Смысл игры (`/meaning`)

Цель страницы:
- Показать глубину: смех как инструмент коммуникации.

Смысловые блоки:
- Ключевая идея продукта.
- Чем подход отличается от обычных «шуток ради шуток».
- Практический результат для игроков/группы.

Контент из источников:
- SEO business description из страницы `meaning`.
- Формулировки из `RJAKA_CORE.md`.

## 5) Карты роли (`/game-card`)

Цель страницы:
- Каталог персонажей для выбора и вдохновения на сценки.

Смысловые блоки:
- Список ролей.
- CTA: «Получить сценку в чате».

База ролей (из источника):
- Полицейский
- Старик ЧО
- Инста Самка
- Папа
- Директор
- Блогер
- Ведущий
- Психолог
- Пират
- Таксист
- Учитель
- Ребенок
- Кот
- Фанат
- Мама
- Робот

## Общие требования ко всем страницам

- Единый стиль через общий CSS: `assets/styles/rjaka-shared.css`.
- Единая навигация: главная, чат, история, 5 вложенных страниц.
- Все страницы публично доступны на временном домене `app.gtstor.com`.
- При переносе на `rjaka.pro` меняется только корневой домен, пути сохраняются.
