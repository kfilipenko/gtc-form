# RJAKA — Unified Design Standard

Единый темный «бархатный» стандарт для всех страниц RJAKA. Палитра, типографика, блоки, адаптив.

## Палитра (CSS custom properties)
- `--raspberry: #B0164D`
- `--magenta: #E34A8A`
- `--black: #0E0610`
- `--gold: #F2B94B`
- `--turquoise: #2FA4A9`
- `--mask: #D6D6D2`
- `--text: #f7f2f6`
- `--text-soft: #e5dce2`
- Фоны: `--bg1`/`--bg2` акценты, `--bg3`/`--bg4` база бархата.
- Градиенты: `--shellGradA/B/C`, `--sectionA/B`.
- Обводки/тени: `--stroke`, `--stroke2`, `--shadowShell`, `--shadowSection`, `--shadowPill`, `--shadowInfo`.

## Типографика
- Заголовки: Playfair Display 600/700.
- Текст: Space Grotesk 400/500/600.
- Заголовки в тоне: h1 малиновый/золото, h2 золото, h3 бирюза.

## Базовая сетка/отступы
- Контейнер `.shell`: ширина `min(1120px, 100%)`, радиус 28px, градиент `shellGradA→B→C`, тень `shadowShell`, padding `--padShell`.
- Тело: фон радиальные пятна + тёмный градиент, padding `--padBody`, центрирование по горизонтали.
- Секции `.section`: радиус 18px, фон `sectionA→sectionB`, бордер `--stroke`, тень `shadowSection`, padding `--padSection`, отступ снизу 12–20px.

## Компоненты
- `.hero`: флекс, wrap, gap 14–28px, выравнивание space-between; внутри h1 + лид + бейдж.
- `.badge`: градиент золото→бирюза, капсула, shadowPill; можно ставить 100% ширины на мобайле.
- `.pill`: капсула-заголовок секции, градиент золото→бирюза, shadowPill.
- `.infobox`: заливка мягкий золото/бирюза, shadowInfo; для формул/инфографики.
- `.quote`: полупрозрачный блок с dashed/solid бордером, курсив.
- `.grid3`: грид 3 колонки (2 на ≤960px, 1 на ≤720px); карточки `.card` с радиусом 16px, бордер `--stroke2`, лёгкая заливка и тень.
- Текстовые утилиты: `.muted` для вторичного текста.

## Адаптив
- ≤960px: grid3 → 2 колонки.
- ≤720px: grid3 → 1 колонка, уменьшение паддингов body/shell.
- ≤560px: h1 уменьшается, badge на всю ширину.

## Быстрый каркас (HTML-структура)
- Подключить шрифты Playfair + Space Grotesk.
- В body: `.shell` → `.hero` → секции `.section` с `.pill`, `.infobox`, `.grid3` и карточками.
- Использовать объявленные CSS переменные и фоновые градиенты из этого стандарта.

## Применение к страницам
- Pages с правилами, промпт-блоками, лендингами: переиспользовать `.shell`, `.hero`, `.section`, `.grid3`, `.infobox`, `.quote`, `.pill`.
- Сохранять палитру и тени; не разбавлять третьими цветами.
- Для форм/чатов: фон, шрифты и капсульные кнопки остаются теми же; акцент кнопок — градиент золото→бирюза или золото→малина.

## Не делать
- Не использовать светлые фоны вне карточек/quote.
- Не менять стек шрифтов на системный.
- Не добавлять серый бордер с высокой непрозрачностью — только `--stroke/--stroke2`.

## Фрагмент CSS (вставка)
Используйте блок `:root`, фон body и стили `.shell`, `.section`, `.pill`, `.infobox`, `.grid3`, `.badge`, как в этом документе. Можно копировать CSS целиком из шаблона и подставлять контент.
